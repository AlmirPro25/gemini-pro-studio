# 🎯 Guia Rápido - Novas Funcionalidades

## 📎 Anexar Arquivos

### Método 1: Clique no Botão
1. Clique no ícone de **clipe** 📎 no input
2. Selecione uma ou mais imagens
3. Veja o preview dos arquivos
4. Digite seu prompt
5. Envie!

### Método 2: Drag & Drop (NOVO!)
1. Arraste arquivos do seu computador
2. Solte sobre o input de texto
3. Veja o preview
4. Envie!

### Método 3: Ctrl+V / Cmd+V (NOVO!)
1. Copie uma imagem (screenshot, print, etc.)
2. Cole com **Ctrl+V** (ou Cmd+V no Mac)
3. A imagem aparece automaticamente
4. Envie!

**Formatos aceitos:** JPG, PNG, WebP, GIF, PDF

## 📷 Capturar Mídia

### Abrir o Modal
Clique no ícone de **câmera** 📷 no input

### 3 Modos Disponíveis:

#### 1. 📤 Upload
- Clique no botão ou arraste imagens
- Selecione múltiplas imagens de uma vez
- Preview antes de enviar

#### 2. 📷 Câmera
- Permite acesso à câmera do dispositivo
- Tire quantas fotos quiser
- Cada foto é adicionada à lista
- Preview em tempo real

#### 3. 🖥️ Screenshot
- Captura a tela do seu computador
- Útil para mostrar:
  - Erros de código
  - Designs
  - Documentos
  - Qualquer coisa na tela

### Enviar
- Veja todas as imagens capturadas
- Remova as que não quer
- Clique em "Adicionar ao Chat"
- As imagens são enviadas automaticamente com um prompt

## 🎨 Modelo Gratuito de Imagens

### Flash 2.0 Experimental (GRÁTIS)
**Modelo experimental com suporte a imagens - Totalmente gratuito!**

### Como Usar

#### Gerar Imagem do Zero
1. Selecione **"Flash 2.0 Experimental (GRÁTIS)"** no dropdown
2. Digite o prompt:
   ```
   Um gato astronauta flutuando no espaço, estilo cartoon colorido
   ```
3. Escolha o aspect ratio (16:9, 1:1, etc.)
4. Clique em Gerar
5. Aguarde ~10 segundos

#### Editar Imagem Existente
1. Selecione **"Flash 2.0 Experimental (GRÁTIS)"**
2. Anexe a imagem que quer editar
3. Digite o que quer mudar:
   ```
   Mude o fundo para um pôr do sol
   ```
4. Envie (não precisa escolher aspect ratio)
5. A IA edita a imagem!

#### Usar Múltiplas Referências
1. Anexe 2-3 imagens de referência
2. Digite:
   ```
   Combine o estilo da primeira imagem com os elementos da segunda
   ```
3. A IA cria algo novo baseado nas referências!

## 🧙‍♂️ Meta-Persona AI

### O que é?
Um **Master AI** que cria especialistas perfeitos sob demanda.

### Como Usar

#### Criar Especialista Individual
1. Clique em **"Meta-Persona"** no header (varinha mágica ✨)
2. Escolha **"Criar Especialista"**
3. Descreva o que precisa:
   ```
   Preciso de um especialista em otimização de algoritmos de 
   machine learning para reduzir tempo de treinamento em 50%
   ```
4. Clique em **"Criar Especialista"**
5. Aguarde ~5 segundos
6. O Master AI cria o especialista perfeito!
7. Clique nele para usar

#### Criar Equipe
1. Escolha **"Criar Equipe"**
2. Descreva o problema complexo:
   ```
   Preciso desenvolver uma arquitetura de microserviços 
   escalável com foco em segurança e performance
   ```
3. Escolha o tamanho da equipe (2-5)
4. O Master AI cria especialistas complementares:
   - Arquiteto de Microserviços
   - Especialista em Segurança
   - Engenheiro de Performance
   - DevOps Engineer
   - Observability Expert
5. Use qualquer um deles!

#### Sugerir Especialistas
1. Escolha **"Sugerir"**
2. O Master AI analisa sua conversa
3. Sugere 3 especialistas relevantes
4. Escolha um para explorar o tópico

### Ver Detalhes do Especialista
- Banner aparece no topo do chat
- Clique na seta para expandir
- Veja:
  - Domínio de expertise
  - Capacidades principais
  - Estilo de comunicação
  - Por que foi criado

## 💡 Dicas e Truques

### Múltiplas Imagens
- Gemini aceita até ~16 imagens por mensagem
- Use para:
  - Comparar designs
  - Mostrar sequência de passos
  - Analisar múltiplos erros
  - Criar colagens

### Edição Iterativa
1. Gere uma imagem
2. Anexe ela de volta
3. Peça modificações
4. Repita até ficar perfeito!


## 🚨 Solução de Problemas

### "Câmera não funciona"
- Verifique permissões do navegador
- Precisa de HTTPS em produção
- Alguns navegadores bloqueiam por padrão

### "Screenshot não aparece"
- Funciona apenas em desktop
- Não funciona em mobile
- Precisa permitir compartilhamento de tela

### "Imagem muito grande"
- Máximo ~20MB por imagem
- Use ferramentas online para comprimir
- Ou tire screenshot em resolução menor

### "Drag & drop não funciona"
- Certifique-se de soltar sobre o input
- Alguns navegadores podem bloquear
- Use o botão de clipe como alternativa

### "Paste não funciona"
- Funciona apenas com imagens copiadas
- Não funciona com arquivos copiados
- Precisa ter foco no input
