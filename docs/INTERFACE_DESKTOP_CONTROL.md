## 🎯 Fluxo de Uso Visual

### Fluxo 1: Ação Inteligente
```
1. Usuário                    2. Sistema                   3. Resultado
┌──────────────┐             ┌──────────────┐            ┌──────────────┐
│ Digite ação  │────────────>│ Analisa tela │───────────>│ ✅ Executado │
│ "Abrir       │             │ com Gemini   │            │              │
│  Chrome"     │             │ AI           │            │ Screenshot   │
└──────────────┘             └──────────────┘            └──────────────┘
```

### Fluxo 2: Visão AI
```
1. Usuário                    2. Sistema                   3. Resultado
┌──────────────┐             ┌──────────────┐            ┌──────────────┐
│ "Encontrar   │────────────>│ Detecta      │───────────>│ ✅ Clicado   │
│  botão OK"   │             │ objeto       │            │              │
│              │             │ Calcula pos. │            │ Posição:     │
│              │             │ Clica        │            │ (500, 300)   │
└──────────────┘             └──────────────┘            └──────────────┘
```

### Fluxo 3: Controle Manual
```
1. Usuário                    2. Sistema                   3. Resultado
┌──────────────┐             ┌──────────────┐            ┌──────────────┐
│ X: 500       │────────────>│ Move mouse   │───────────>│ ✅ Movido    │
│ Y: 300       │             │ para (500,   │            │              │
│ [Clicar]     │             │ 300)         │            │ Clicado      │
│              │             │ Clica        │            │              │
└──────────────┘             └──────────────┘            └──────────────┘
```
