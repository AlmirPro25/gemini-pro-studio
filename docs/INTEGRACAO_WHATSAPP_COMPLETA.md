## 🚀 Como Usar

### Instalação Rápida

```bash
# 1. Instalar dependências
cd whatsapp-bridge
npm install

# 2. Configurar
cp .env.example .env
# Edite .env e adicione sua GEMINI_API_KEY

# 3. Iniciar
npm start

# 4. Escanear QR Code com WhatsApp
# 5. Pronto! ✅
```
